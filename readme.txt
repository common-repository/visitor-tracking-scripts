== Description ==

Add your Google Analytics code or other visitor tracking script to your site. 

After activating the plugin go to Settings -> Visitor Tracking and enter your Google Analytics code or paste the tracking script of some other service you are using.

This plugin is contained in the free Poor Man's [WordPress SEO pack](http://mokimoki.net/poor-mans-wordpress-seo/)  that has also a lot of other important SEO features.
